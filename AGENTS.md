# Expo HAS CHANGED

Read the exact versioned docs at https://docs.expo.dev/versions/v54.0.0/ before writing any code.

## Project Rules

- 作業前に必ず `CLAUDE.md` を読み、記載のルール・技術スタック・制約に従うこと。
- `docs/GOTCHAS.md` が存在する場合は必ず読み、禁止事項・過去の手戻り事例に従うこと。
- 設計書・既存実装からの逸脱、文言の要約・省略・言い換えをしないこと。
- 判断に迷う分岐があれば、実装を進めずに「要確認事項」として最終報告に明記すること（独自判断で進めないこと）。
- 完了後は変更ファイル一覧・`git diff --stat`・確認方法を報告すること。
- **セッション分割**：1タスク完了を1セッションの区切りとする。完了後は ProgressLog（`docs/progress_log.md`）に完了済み・未完了・判断の経緯を書き出してPMへ報告し、セッションを終了すること。次タスクは新セッションで `CLAUDE.md` + ProgressLog を読んでから開始。コンテキスト圧縮が発生した場合も `CLAUDE.md` を即再読み込みして状態を復元すること。
