## 1. 使用禁止の独自文言リスト

以下は過去に独自実装され、設計書逸脱として修正指示が出た文言です。
**二度と使用しないこと。**

| 禁止文言 | 正しい文言（設計書準拠） | 発生箇所 |
|---|---|---|
| いままでのストップ貯金（推定） | 今月の推定節約額 / 累計推定節約額 | ホーム画面・節約カウンター |
| 税込みの金額を入力（プレースホルダー） | プレースホルダーなし（設計書に記載なし） | 3問チェックQ2金額欄 |
| 節約カウンターを見る（設定画面の独自項目） | 設定画面に記載なし。削除対象 | 設定画面 |
| 「サポーターになる」 | 使用しない（PMから話なしで決定済み） | 全画面共通 |

**なぜ起きたか：** デザイン資料（quiet-ledgerフォルダのSS）の文言を、Codexが設計書より優先して実装したため。

## 2. 設計書とデザイン資料が矛盾している箇所

`assets/design1.png` および `reports/screenshots/quiet-ledger/` のSSは
**UIビジュアルの参考のみ**。文言・機能仕様は設計書が最優先。

確認された矛盾点：
- ホームのサブコピー：設計書「買う前に、3問だけ。」 vs デザイン資料「3つの質問で、いまの自分を記録しよう」
- Q1/Q3の入力方式：設計書「セグメントボタン」 vs デザイン資料「テキスト自由入力」
- Q2金額の入力方式：設計書「レンジ選択」 vs デザイン資料「テキスト自由入力（税込み）」
- 節約額の文言・有料ロックの有無：上記表のとおり
- 履歴ロック画面の文言・価格表示スタイル：「価格: 250円」(設計書) vs 「250円 買い切り」+説明文（デザイン資料採用・PM承認済み 2026-06-05）

→ **矛盾を見つけたら独自判断で採用せず、必ずPMに報告すること。**
→ PMが「デザイン資料の○○を採用する」と明示した場合のみ採用可。
